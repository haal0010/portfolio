A little bit about me

Welcome to my portfolio! I’m Hafsa, a passionate web designer with a focus on creating clean, user-friendly websites. I blend creativity with technical skills to deliver seamless digital experiences that engage and inspire. Throughout my journey as a web design student, I’ve honed my abilities in HTML, CSS, and JavaScript, and I’m excited to showcase the projects, I’ve worked on during my first semester.

Skills

Portfolio

  • Web

Learned the fundamentals of web development by creating domains, uploading files, and building a website from the ground up. The process involved designing a mobile-first layout using HTML and CSS, incorporating media queries, grid systems, and flexbox. Challenges included getting acquainted with new tools like FileZilla, Figma, and VS Code.

  • UX/UI

Focused on design by researching the target audience and experimenting with moodboards, style tiles, and wireframes. The process included both low-fidelity and high-fidelity prototyping in Figma, ensuring the site was not only visually appealing but also functional. Key learning points were creating interactive elements and optimizing assets for performance.

  • Animation

Explored interactive storytelling by sketching game prototypes and transitioning those ideas into digital animations. This theme involved brainstorming on paper, implementing CSS keyframe animations, and integrating JavaScript for interactivity. Using Adobe Illustrator for scalable SVG graphics and diagramming the animation flow with tools like draw.io were essential parts of the learning curve.

  • Blue Cafe

Collaborated on a comprehensive redesign of Café Blue’s website by combining research, brainstorming, and iterative design processes. This project involved creating moodboards, site maps, wireframes, and prototypes, as well as coding interactive features like a scrollable gallery. Working with git for version control and conducting usability tests (including Likert and Lighthouse tests) highlighted the benefits and challenges of teamwork in web development..
